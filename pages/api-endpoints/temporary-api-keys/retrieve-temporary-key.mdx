---
title: "Retrieve API key"
api: "GET https://api.keywordsai.co/api/temporary-keys/{{temporary_api_key_id}}/"
---

You can retrieve a temporary API key by sending a GET request to the temporary keys endpoint.

## Path parameters

<ParamField path="temporary_api_key_id" type="string">
  The ID of the temporary API key to retrieve.
</ParamField>
