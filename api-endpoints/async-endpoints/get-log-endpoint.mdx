---
title: Get log
api: GET https://api.keywordsai.co/api/request-logs/{unique_id}
---

The Get Log endpoint allows you to get a specific log by its unique ID.

<Note>
You should first get the log list, and based on the unique id in the list, retrieve the specific log. Learn more about the [log list endpoint](/api-endpoints/async-endpoints/logs-list-endpoint).
</Note>

## Path params  

<ParamField path="unique_id" type="string" required>
  The unique ID of the log to get.
</ParamField>

```python
import requests

url = "https://api.keywordsai.co/api/request-logs/{{unique_id}}/"

payload = ""
headers = {
  'Authorization': 'Bearer {{your_api_key}}'
}

response = requests.request("GET", url, headers=headers, data=payload)

print(response.text)
```
