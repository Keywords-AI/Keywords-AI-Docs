---
title: "List customers"
api: "GET https://api.keywordsai.co/api/users/"
---

Returns a paginated list of customers in your organization.

## Authentication

<Info>
Use either:

- `Authorization: Bearer YOUR_API_KEY` (public API)
- `Authorization: Bearer YOUR_JWT_TOKEN` (platform/internal)
- `X-Auth-Api-Key: YOUR_API_KEY` (alternative header)
</Info>

## Query parameters

<ParamField path="page" type="integer" default={1}>
Page number.
</ParamField>

<ParamField path="page_size" type="integer" default={100}>
Number of customers per page. Maximum: `1000`.
</ParamField>

<ParamField path="environment" type="string" default="prod">
Filter by environment. Options: `prod`, `test`.
</ParamField>

<ParamField path="search" type="string">
Search by `customer_identifier`, `email`, or `name`.
</ParamField>

<ParamField path="has_budget" type="boolean">
Filter customers that have budgets set.
</ParamField>

## Response fields

<ParamField name="count" type="integer">
Total number of customers matching the query.
</ParamField>

<ParamField name="next" type="string">
URL to the next page (or `null`).
</ParamField>

<ParamField name="previous" type="string">
URL to the previous page (or `null`).
</ParamField>

<ParamField name="results" type="array">
List of customer objects.
<Accordion title="Customer object fields">
<ParamField name="customer_identifier" type="string">
Your unique identifier for this customer.
</ParamField>
<ParamField name="email" type="string">
Customer email (optional).
</ParamField>
<ParamField name="name" type="string">
Customer display name (optional).
</ParamField>
<ParamField name="environment" type="string">
Environment (`prod` or `test`).
</ParamField>
<ParamField name="period_budget" type="number">
Budget for the current period in USD (or `null`).
</ParamField>
<ParamField name="budget_duration" type="string">
Budget renewal period (`daily`, `weekly`, `monthly`).
</ParamField>
<ParamField name="period_usage" type="number">
Total usage in the current period in USD.
</ParamField>
<ParamField name="total_cost" type="number">
All-time total usage in USD.
</ParamField>
<ParamField name="total_logs" type="integer">
All-time total number of requests.
</ParamField>
<ParamField name="last_active" type="string">
ISO timestamp of the last request.
</ParamField>
</Accordion>
</ParamField>
