---
title: "User detail"
api: "GET https://api.keywordsai.co/api/users/<customer_identifier>/"
---

Retrieves detailed information about a specific customer, including current budget usage.

## Path parameters

<ParamField path="customer_identifier" type="string" required={true}>
Your unique identifier for this customer.
</ParamField>

## Query parameters

<ParamField path="environment" type="string" default="prod">
Specify environment if you are not using the default. Options: `prod`, `test`.
</ParamField>

## Example call
<CodeGroup>
```Python Python
import requests

url = "https://api.keywordsai.co/api/users/user_123/"
api_key = "YOUR_API_KEY"
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```TypeScript TypeScript
fetch("https://api.keywordsai.co/api/users/user_123/", {
  method: "GET",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
  }
})
.then(response => response.json())
.then(data => console.log(data));
```
</CodeGroup>
