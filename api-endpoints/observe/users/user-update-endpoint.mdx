---
title: "User update"
api: "PATCH https://api.keywordsai.co/api/users/<customer_identifier>/"
---

Updates an existing customer. Use this to change budgets or contact info.

## Path parameters

<ParamField path="customer_identifier" type="string" required={true}>
Your unique identifier for this customer.
</ParamField>


## Request body (any subset)

<ParamField path="email" type="string">
Customer email address.
</ParamField>

<ParamField path="name" type="string">
Customer display name.
</ParamField>

<ParamField path="period_budget" type="number">
Budget for the current period in USD. Set to `null` to remove the budget limit.
</ParamField>

<ParamField path="budget_duration" type="string">
Budget renewal period. Options: `daily`, `weekly`, `monthly`.
</ParamField>

<RequestExample>
```Python Python
import requests

url = "https://api.keywordsai.co/api/users/user_123/"
api_key = "YOUR_API_KEY"
data = {
    "period_budget": 200.0
}
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.patch(url, headers=headers, json=data)
print(response.json())
```

```TypeScript TypeScript
fetch("https://api.keywordsai.co/api/users/user_123/", {
  method: "PATCH",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
  },
    body: JSON.stringify({
        period_budget: 200.0,
    })
})
.then(response => response.json())
.then(data => console.log(data));
```

</RequestExample>

<Tip>
To remove a budget limit, set `period_budget` to `null`.
</Tip>
