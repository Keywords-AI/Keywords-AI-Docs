# List Evaluators

**GET** `/api/evaluators/`

Retrieves a paginated list of evaluators for your organization.

## Authentication

Requires API key authentication. Include your API key in the request headers:

```bash
Authorization: Api-Key YOUR_API_KEY
```

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `page` | integer | Page number for pagination (default: 1) |
| `page_size` | integer | Number of items per page (default: 20) |
| `search` | string | Search evaluators by name or slug |
| `type` | string | Filter by evaluator type: `llm`, `human`, or `code` |
| `score_value_type` | string | Filter by score type: `numerical`, `boolean`, `categorical`, or `comment` |
| `starred` | boolean | Filter by starred status |
| `tags` | string | Filter by tags (comma-separated) |

## Examples

<CodeGroup>

```python Python
import requests

url = "https://api.keywordsai.co/api/evaluators/"
headers = {
    "Authorization": "Api-Key YOUR_API_KEY"
}

# Basic list
response = requests.get(url, headers=headers)
print(response.json())

# With filters
params = {
    "type": "llm",
    "score_value_type": "numerical",
    "page_size": 10
}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```

```bash cURL
# Basic list
curl -X GET "https://api.keywordsai.co/api/evaluators/" \
  -H "Authorization: Api-Key YOUR_API_KEY"

# With filters
curl -X GET "https://api.keywordsai.co/api/evaluators/?type=llm&score_value_type=numerical&page_size=10" \
  -H "Authorization: Api-Key YOUR_API_KEY"
```

</CodeGroup>

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `results` | array | Array of evaluator objects |

| `next` | string\|null | URL for next page (if exists) |
| `current_filters` | object | Applied filters for this request |
