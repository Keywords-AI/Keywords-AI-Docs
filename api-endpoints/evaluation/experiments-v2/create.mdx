---
title: "Create experiment (V2)"
api: "POST https://api.keywordsai.co/api/v2/experiments"
---

Create a new experiment using the V2 API with dataset and workflows.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Request Body

<RequestExample>
```json Request
{
  "dataset_id": "dataset_id_123",
  "name": "My Experiment V2",
  "description": "Testing experiment creation",
  "workflows": [
    {
      "type": "custom_eval_workflow",
      "config": {}
    }
  ],
  "evaluator_slugs": [
    "evaluator_slug_1"
  ]
}
```
</RequestExample>

## Examples

<CodeGroup>

```python Python
import requests

url = "https://api.keywordsai.co/api/v2/experiments"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "dataset_id": "dataset_id_123",
    "name": "My Experiment V2",
    "description": "Testing experiment creation",
    "workflows": [
        {
            "type": "custom_eval_workflow",
            "config": {}
        }
    ],
    "evaluator_slugs": [
        "evaluator_slug_1"
    ]
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```bash cURL
curl -X POST "https://api.keywordsai.co/api/v2/experiments" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "dataset_id": "dataset_id_123",
    "name": "My Experiment V2",
    "workflows": [{"type": "custom_eval_workflow", "config": {}}]
  }'
```

</CodeGroup>
