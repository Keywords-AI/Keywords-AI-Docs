---
title: "Run experiment evals"
api: "POST https://api.keywordsai.co/api/experiments/{experiment_id}/run-evals"
---

Run evaluators on an experiment's results.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Path Parameters

<ParamField path="experiment_id" type="string" required>
  The ID of the experiment to run evaluations on.
</ParamField>

## Request Body

<RequestExample>
```json Request
{
  "evaluator_slugs": [
    "is_english"
  ]
}
```
</RequestExample>

## Examples

<CodeGroup>

```python Python
import requests

experiment_id = "experiment_id_123"
url = f"https://api.keywordsai.co/api/experiments/{experiment_id}/run-evals"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "evaluator_slugs": [
        "is_english"
    ]
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```bash cURL
curl -X POST "https://api.keywordsai.co/api/experiments/{experiment_id}/run-evals" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "evaluator_slugs": ["is_english"]
  }'
```

</CodeGroup>

## Response

<ResponseExample>
```json 200 OK
{
  "status": "running",
  "experiment_id": "experiment_id_123"
}
```
</ResponseExample>
