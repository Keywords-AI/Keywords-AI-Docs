---
title: "Get Experiment Summary"
api: "GET https://api.keywordsai.co/api/v2/experiments/{experiment_id}/logs/summary/"
---

Retrieves aggregated summary statistics for experiment traces matching the current filters. This is useful for getting overview metrics without retrieving individual traces.

## Authentication

All endpoints require API key authentication:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Parameters

### Query Parameters

<ParamField query="start_time" type="string">
  Filter start time (ISO format).
</ParamField>

<ParamField query="end_time" type="string">
  Filter end time (ISO format).
</ParamField>

### Body Parameters (Optional)

For advanced filtering, you can send a POST request with filters in the body (same format as list endpoint).

<ParamField body="filters" type="array">
  List of filters to apply.
</ParamField>

## Response

```json
{
  "total_count": 150,
  "total_cost": 12.45,
  "total_tokens": 50000,
  "total_prompt_tokens": 30000,
  "total_completion_tokens": 20000,
  "avg_latency": 1.23
}
```

## Example

```bash
curl -X GET "https://api.keywordsai.co/api/v2/experiments/experiment-123/logs/summary/" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
