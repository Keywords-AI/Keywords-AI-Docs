---
title: "Get Experiment Log Details"
api: "GET https://api.keywordsai.co/api/v2/experiments/{experiment_id}/logs/{trace_id}/"
---

Retrieves detailed information about a specific trace, including full input content and span hierarchy.

## Authentication

All endpoints require API key authentication:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Response

```json
{
  "id": "551ba4023fc646b4859993a31665bff2",
  "name": "Dataset Entry 1",
  "input": "[{\"role\": \"system\", \"content\": \"You are a helpful assistant\"}, {\"role\": \"user\", \"content\": \"What is the capital of France?\"}]",
  "output": "{\"message\": \"Awaiting custom workflow result submission\", \"status\": \"pending_custom_submission\"}",
  "status": "pending_custom_submission",
  "span_count": 1,
  "duration": null,
  "created_at": "2025-11-18T10:01:00Z",
  "span_tree": [
    {
      "id": "4d872fe46b264d0b994b0af1894b4ade",
      "span_name": "experiment_trace",
      "input": "[{\"role\": \"system\", \"content\": \"You are a helpful assistant\"}, {\"role\": \"user\", \"content\": \"What is the capital of France?\"}]",
      "output": "{\"message\": \"Awaiting custom workflow result submission\", \"status\": \"pending_custom_submission\"}",
      "status": "pending_custom_submission",
      "span_type": "ROOT",
      "created_at": "2025-11-18T10:01:00Z"
    }
  ]
}
```

## Example

```bash
curl -X GET "https://api.keywordsai.co/api/v2/experiments/experiment-123/logs/551ba4023fc646b4859993a31665bff2/" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
