---
title: "Update experiment"
api: "PATCH https://api.keywordsai.co/api/experiments/{experiment_id}"
---

Update an experiment's metadata.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Path Parameters

<ParamField path="experiment_id" type="string" required>
  The ID of the experiment to update.
</ParamField>

## Request Body

<RequestExample>
```json Request
{
  "name": "Updated Experiment Name",
  "description": "Updated description"
}
```
</RequestExample>

## Examples

<CodeGroup>

```python Python
import requests

experiment_id = "experiment_id_123"
url = f"https://api.keywordsai.co/api/experiments/{experiment_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "name": "Updated Experiment Name"
}

response = requests.patch(url, headers=headers, json=data)
print(response.json())
```

```bash cURL
curl -X PATCH "https://api.keywordsai.co/api/experiments/{experiment_id}" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Experiment Name"
  }'
```

</CodeGroup>

## Response

<ResponseExample>
```json 200 OK
{
  "id": "experiment_id_123",
  "name": "Updated Experiment Name"
}
```
</ResponseExample>
