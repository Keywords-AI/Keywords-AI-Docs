---
title: "Update experiment rows"
api: "PATCH https://api.keywordsai.co/api/experiments/{experiment_id}/rows"
---

Update rows in an experiment.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Path Parameters

<ParamField path="experiment_id" type="string" required>
  The ID of the experiment to update rows in.
</ParamField>

## Request Body

<RequestExample>
```json Request
{
  "rows": [
    {
      "id": "074ca0b7b28c447996ad7447204b5910",
      "input": {
        "current_prompt_version": "v2",
        "prompt_versions_diff": "come on, some big diff here"
      }
    },
    {
      "input": {
        "current_prompt_version": "v3",
        "prompt_versions_diff": "more_diffs"
      }
    }
  ]
}
```
</RequestExample>

## Examples

<CodeGroup>

```python Python
import requests

experiment_id = "experiment_id_123"
url = f"https://api.keywordsai.co/api/experiments/{experiment_id}/rows"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "rows": [
        {
            "id": "row_id_123",
            "input": {
                "current_prompt_version": "v2"
            }
        }
    ]
}

response = requests.patch(url, headers=headers, json=data)
print(response.json())
```

```bash cURL
curl -X PATCH "https://api.keywordsai.co/api/experiments/{experiment_id}/rows" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "rows": [{"id": "row_id", "input": {}}]
  }'
```

</CodeGroup>
