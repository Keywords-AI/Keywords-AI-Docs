---
title: "Add experiment rows"
api: "POST https://api.keywordsai.co/api/experiments/{experiment_id}/rows"
---

Add rows to an experiment.

## Authentication

- API key: `Authorization: Bearer <API key>`

## Path Parameters

<ParamField path="experiment_id" type="string" required>
  The ID of the experiment to add rows to.
</ParamField>

## Request Body

<RequestExample>
```json Request
{
  "rows": [
    {
      "input": {
        "current_prompt_version": "v2",
        "prompt_versions_diff": "some_other_diff"
      }
    },
    {
      "input": {
        "current_prompt_version": "v3",
        "prompt_versions_diff": "more_diffs"
      }
    }
  ]
}
```
</RequestExample>

## Examples

<CodeGroup>

```python Python
import requests

experiment_id = "experiment_id_123"
url = f"https://api.keywordsai.co/api/experiments/{experiment_id}/rows"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "rows": [
        {
            "input": {
                "current_prompt_version": "v2"
            }
        }
    ]
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```bash cURL
curl -X POST "https://api.keywordsai.co/api/experiments/{experiment_id}/rows" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "rows": [{"input": {}}]
  }'
```

</CodeGroup>

## Response

<ResponseExample>
```json 200 OK
{
  "added": 2,
  "rows": []
}
```
</ResponseExample>
