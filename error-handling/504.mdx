---
title: "504: Server Timeout"
---

## Description

This indicates the Keywords API server is currently overloaded and therefore cannot respond in time. This is a temporary condition and will be resolved once the server load decreases.


<AccordionGroup>
  <Accordion title="Timeout">
    Keywords AI is too busy to respond to your request.
    
    **Possible Solutions**
    - Decrease the number of your concurrent requests. It is recommended to have a maximum of 300 concurrent requests.
    - Wait a few seconds and try again.
  </Accordion>
</AccordionGroup>
