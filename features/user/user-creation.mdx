To have better user observability, you can create new users through our [User Creation endpoint](/api-endpoints/async-endpoints/user-creation-endpoint) or update exsiting users through [User Update endpoint](/api-endpoints/async-endpoints/user-update-endpoint). 

## User creation
You have 2 ways to create users on Keywords AI: through API call with the inference/logging endpoint or through the User Creation endpoint.

<Note>Use the [User Creation endpoint](/api-endpoints/async-endpoints/user-creation-endpoint) instead of our other endpoints</Note>
```
url = "https://api.keywordsai.co/api/users/create/"
```
<CodeGroup>
```Python Python
import requests

url = "https://api.keywordsai.co/api/users/create/"
api_key = "YOUR_KEY" # Replace with your actual Keywords AI API key
data = {
    "customer_identifier": "some_customer_identifier",
    "name": "John Doe",
    # Add other parameters here 
    # e.g. "period_budget": 30
}
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

```

```TypeScript TypeScript
// Define the function with TypeScript
fetch('https://api.keywordsai.co/api/users/create/', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer Your_Keywords_AI_API_Key '
  },
    body: JSON.stringify({
        customer_identifier: "some_customer_identifier"
        name: "John Doe",
        // Add other parameters here 
        // e.g. period_budget: 30
    })
})
.then(response => response.json())
.then(data => console.log(data));
```
</CodeGroup>

For more inforation about `customer_identifier`, check out the parameter [here](/features/generation/customer-identifier)
