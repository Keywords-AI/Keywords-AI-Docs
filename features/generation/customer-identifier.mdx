---
title: "Customer identifier"
description: "Use this as a tag to identify the user associated with the API call."
icon: "users"
---

After you pass the `customer_identifier` parameter in your API call, you can monitor the user's data in the Keywords AI platform. 
See how to get insights into your users' data [here](/features/user/user-data)

## Example code
```json
{
    //...other_params,
    "customer_identifier": "user_123"
}

```

## Detailed example
<CodeGroup>
```python Python
import requests

def  keywords_ai_generate( 
                messages, 
                api_key=None,
                **kwargs
                ):

    headers = {
        'Content-Type': 'application/json',
        'Authorization': f'Api-Key {api_key}',
    }
    data = {
        "messages": messages,
        **kwargs
    }
        
    response = requests.post(
        'https://api.keywordsai.co/api/chat/completions', headers=headers, json=data)
    return response

response = keywords_ai_generate(
    messages=[{"role":"user", "content":"Hi"}], 
                     model="gpt-3.5-turbo", 
                     api_key="En5XoPkf.kSE2edr33UCjttnqhCzlN5tz5niosedg",
                     # Add the customer_identifier
                    customer_identifier="customer_1"
              )
```
