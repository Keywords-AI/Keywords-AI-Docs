## How to create a dataset
1. You could create a dataset from the logs page by clicking the `Create dataset` button.
2. Apply filters to the logs to create a dataset.
3. You could also handpick logs you want to include in the dataset.
4. Delete logs from the dataset by clicking the checkbox of the logs and clicking the `Delete` button.
5. Export the dataset to a CSV/json file by clicking the `Export` button.
