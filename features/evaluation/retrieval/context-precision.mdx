## Definition
Context Precision is used to measure information density.

<img src="/images/evaluation/retrieval/context-precision-definition.png"/>

See more at [Relari AI's definition](https://docs.relari.ai/v0.3/metrics/retrieval/llm-based/llm_context_precision)

## Settings and parameters

1. Go to Keywords AI (on top of the left nav bar) > Evaluation > Retrieval > Context Precision
<img src="/images/evaluation/retrieval/context-precision-setting.png"/>

2. Click on the card to create the setting:
<img src="/images/evaluation/retrieval/context-precision-modal.png"/>
- Click the enable switch to turn on the evaluation
- Pick a LLM model you want to run the evaluation with
- Hit the "Save" button.

3. Make an API call, and the evaluation will be run based on the [`Ramdom sampling`](/features/evaluation/overview#sampling) setting.
