---
title: Sentiment analysis
---

Sentiment Analysis inspects the given text and identifies the prevailing emotional opinion within the text, especially to determine a writer's attitude as positive, negative, or neutral. Sentiment contains the sentence level sentiment values attached to each sentence, which contain score between -1.0 (negative) and 1.0 (positive).

## Settings and parameters

1. Go to Keywords AI (on top of the left nav bar) > Evaluation > Other > Sentiment
<img src="/images/evaluation/other/sentiment-settings.png"/>

2. Click on the `Sentiment` card to create the setting:
<img src="/images/evaluation/other/sentiment-modal.png"/>
- Click the enable switch to turn on the evaluation

3. Make an API call, and the evaluation will be run based on the [`Ramdom sampling`](/features/evaluation/overview#sampling) setting.

4. Check results in the [requests log](https://platform.keywordsai.co/platform/requests)

<img src="/images/evaluation/check-results.png" />
