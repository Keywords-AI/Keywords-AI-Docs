## Overview
The full-text search system lets you search logs by any text in the log messages. You can search text in the System prompt, User prompt and Completion.

## You will love it when you want to:
- Find a specific log by any text in the log messages.
- Save time by searching for logs you need.

## Quick start
Click the `Filter` button to open the filter panel. Scroll down to the `Messages` section. Click `Messages` and type the text you want to search in the popup window.
