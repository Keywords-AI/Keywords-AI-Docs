## Install OpenAI SDK for Python

Keywords AI is completely compatible with OpenAI’s Python library for ease of code migration

`pip install openai`

## Try Keywords AI with Python SDK

```python
import openai
client = openai.OpenAI()
openai.base_url = "https://platform.keywordsai.co/api/generate/"
openai.api_key = "{YOUR_KEYWORDSD_AI_API_KEY}" # You should replace this with your actual key
print(client.chat.completions.create(
    model = "gpt-4-0613",
    messages = [{"role": "user", "content": "Say 'Hello World'"}]))
```

- **URL Change**: Modify the API endpoint URL in your code from OpenAI’s URL to the Keywords AI endpoint URL: \`https://keywordsai.co/api/generate/\\`.
- **API Key**: Replace the OpenAI API key with your Keywords AI API key.
- **Parameters**: Models parameter becomes optional.
