## Try Keywords AI with Python SDK

```python
def demo_call(input, 
              model="claude-2" ,
              token="R2iZMDRj.r0vZMKKEV40PBR0nPsIrm9O6xJQaqwnd", 
              stream=False
              ):
    headers = {
        'Content-Type': 'application/json',
        'Authorization': f'Api-Key {token}',
    }

    data = {
        'model': model,
        'messages': [{'role': 'user', 'content': input}],
        "stream": stream,
    }

    response = requests.post('https://platform.keywordsai.co/api/generate/', headers=headers, json=data, stream=stream)
    # response = requests.post('http://localhost:8000/api/generate/', headers=headers, json=data)
    return response

messages = [{'role': 'user', 'content': 'Hello, how are you?'}]
print(demo_call(messages).json())
```

- **URL Change**: Modify the API endpoint URL in your code from OpenAI’s URL to the Keywords AI endpoint URL: \`https://keywordsai.co/api/generate/\\`.
- **API Key**: Replace the OpenAI API key with your Keywords AI API key.
- **Parameters**: Models parameter becomes optional.
