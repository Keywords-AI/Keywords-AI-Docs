---
title: "Anthropic"
description: "Use your own Anthropic credits through Keywords AI"
---

Add `customer_credentials` parameter in your [request body](/api-endpoints/proxy-endpoints/chat-completions) to use your own Anthropic credits.

```json
{
  // Rest of the request body
  "customer_credentials": {
    "anthropic": {
      "api_key": "your-api-key"
    }
  }
}
```
