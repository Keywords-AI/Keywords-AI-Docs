## Add it in code
- Add `customer_credentials` parameter in your [request body](/api-endpoints/proxy-endpoints/chat-completions) to use your own Perplexity credits.

```json
{
  // Rest of the request body
  "customer_credentials": {
    "perplexity": {
      "api_key": "your-api-key"
    }
  }
}
```
