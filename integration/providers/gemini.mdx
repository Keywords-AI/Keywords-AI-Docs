## Grounding with Google Search

You can enable grounding with Google Search for Gemini/Vertex AI models by adding `googleSearch` parameter to your request body.

```python {9-11}
params = {
    "messages": [
        {
            "role": "user",
            "content": "what is the amazon stock price right now"
        }
    ],
    "model": "gemini/gemini-2.0-flash",
    "tools": [
        {"googleSearch": {}} # Add this tool to enable google search. Only works for gemini/ and vertex_ai/ models
    ]
}  
```
