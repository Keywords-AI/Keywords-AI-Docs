---
title: "Groq"
description: "Use your own Groq credits through Keywords AI"
---

Add `customer_credentials` parameter in your [request body](/api-endpoints/proxy-endpoints/chat-completions) to use your own Groq credits.

```json
{
  // Rest of the request body
  "customer_credentials": {
    "groq": {
      "api_key": "your-api-key"
    }
  }
}
```
