---
title: "OpenAI"
description: "Use your own OpenAI credits through Keywords AI"
---

Add `customer_credentials` parameter in your [request body](/api-endpoints/proxy-endpoints/chat-completions) to use your own OpenAI credits.

```json
{
  // Rest of the request body
  "customer_credentials": {
    "openai": {
      "api_key": "your-api-key"
    }
  }
}
```
