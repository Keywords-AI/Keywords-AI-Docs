## Add it in code
- Add `customer_credentials` parameter in your [request body](/api-endpoints/proxy-endpoints/chat-completions) to use your own OpenRouter credits.

```json
{
  // Rest of the request body
  "customer_credentials": {
    "openrouter": {
      "api_key": "your-api-key"
    }
  }
}
```
