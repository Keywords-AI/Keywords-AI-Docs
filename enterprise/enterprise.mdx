Keywords AI's Enterprise Platform provides enhanced performance and dedicated support for our enterprise customers. This guide will walk you through the setup process and help you integrate with our enterprise endpoints.

## 1. Sign up for enterprise platform
Access the Enterprise Platform through either: 
- [https://enterprise.keywordsai.co](https://enterprise.keywordsai.co)
- Keywords AI login page: Select the `Enterprise portal` button.
<Frame>
<img src="https://keywordsai-static.s3.us-east-1.amazonaws.com/docs/ops/enterprise-portal.png" alt="Login page" />
</Frame>

## 2. API endpoint
We provide a separate API endpoint for enterprise customers. 
Use this endpoint: `endpoint.keywordsai.co` to send API requests.
### Example
**Logging API**: 

❌`https://api.keywordsai.co/api/request-logs/create/`
✅`https://endpoint.keywordsai.co/api/request-logs/create/`

Learn more about [Logging API](https://docs.keywordsai.co/get-started/async-logging).

**LLM proxy**

❌`https://api.keywordsai.co/api/chat/completions`
✅`https://endpoint.keywordsai.co/chat/completions`

Learn more about [LLM proxy](https://docs.keywordsai.co/get-started/llm-inference).
